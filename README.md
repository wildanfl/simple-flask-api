# simple-flask-api
Projek ini adalah contoh implementasi sederhana untuk membuat web server CRUD menggunakan Flask dan MySQL Workbench.
Instalasi
Clone repositori ini ke dalam direktori lokal Anda

Buat database baru pada MySQL Workbench dengan nama db_nilai dan tabel mahasiswa. Tabel mahasiswa memiliki kolom-kolom berikut:

NIM (varchar(50))
Nama (varchar(255))
Kelas (varchar(50))
Fundamental_Cloud (int)
Db (int)
Py (int)
Linux (int)
Sesuaikan koneksi ke database pada file app.py dengan informasi login dan host yang sesuai dengan konfigurasi MySQL Workbench Anda.

Jalankan file app.py pada terminal dengan perintah python app.py.

Buka browser dan akses alamat http://localhost:5000.
